This folder contains tiny demo inputs for smoke testing the pipeline.

Files:
- demo_R1.fastq / demo_R2.fastq: small paired-end reads for dry-run and parser tests
- demo_host_reference.fasta: tiny mock host reference used in examples

These files are not biologically meaningful and are only intended to verify wiring.
For real analysis, replace them with sequencing data from your experiment or a public archive such as NCBI SRA.
